Project Phoenix

A personal discipline and life-tracking system built to convert daily actions into data, clarity, and long-term self-mastery.

Built by Jogeshwar Mishra · MERN Stack

What is Project Phoenix?

Project Phoenix is a personal life-operating system designed to track, trace, and transform daily behavior into disciplined execution. This is not a public productivity app. It is built for one purpose: eliminating inconsistency and excuses.

Core Philosophy

  • Motivation fades. Systems stay.
  • Memory lies. Logs don’t.
  • Feelings confuse. Data clarifies.

Core Modules

Daily Life Tracker

Tracks wake-up time, sleep, office hours, study time, exercise, meals, mood, and energy levels to answer one question: what actually happened today?

Discipline & Rules Engine

Custom self-rules with streak tracking and violation logging. Exposes where promises to self are broken.

Study & Career Tracker

Tracks subjects, topics, time invested, and confidence levels. Weekly analysis highlights weak areas and false productivity.

Failure Tracing System

Records the real reasons behind bad days and repeated failures, converting mistakes into feedback instead of guilt.

Analytics Dashboard

Visual insights into discipline score, consistency, mood vs productivity, and workload distribution.

Technology Stack

Frontend: React (Vite)
Backend: Node.js + Express
Database: MongoDB
Authentication: JWT
UI: Minimal dark theme
Deployment: Vercel / Render
